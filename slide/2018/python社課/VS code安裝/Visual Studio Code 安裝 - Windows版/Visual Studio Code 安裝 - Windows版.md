# Visual Studio Code 簡易教學

---

在桌面隨便新增一個資料夾

然後對資料夾點右鍵

你應該會看到**Open with Code**

點他

---

![](https://i.imgur.com/YoIlWso.png =1000x)

---

進去後

在左邊空空的那邊按右鍵

選**新增檔案**

然後打你要的檔名

---

![](https://i.imgur.com/BI16fhF.png =1000x)

---

然後你就可以開始打程式了

![](https://i.imgur.com/sR3V7b9.png)
